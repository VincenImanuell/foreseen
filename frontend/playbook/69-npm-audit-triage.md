# Npm Audit Triage

Treat npm audit output as triage input for the frontend dependency graph.

Checklist:

- Keep the note tied to the Foreseen game frontend and the active Celo network.
- Do not promise a package bump is safe without a build and route check.
- Record whether the issue affects production browser code or dev tooling only.
- Leave secrets, generated builds, and local caches out of Git.
