# MiniPay CELO Session

Use this note when reviewing a MiniPay build during an active CELO campaign.

Checklist:

- Verify MiniPay auto-detects the injected wallet before testing match actions.
- Confirm the selected network matches `NEXT_PUBLIC_CELO_NETWORK`.
- Keep Sepolia test sessions separate from mainnet CELO stake sessions.
- Check that wallet return states do not hide reveal or withdraw actions.
- Record UI findings without adding secrets, screenshots with keys, or generated build output.
