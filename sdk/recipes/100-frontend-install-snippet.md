# Frontend Install Snippet

Use this recipe when a consumer wants to install the SDK inside a frontend app.

Checklist:

- Show the npm package name as `@foreseen/sdk`.
- Keep the install command as `npm install @foreseen/sdk` unless a package manager section says otherwise.
- Verify the snippet imports from the package entrypoint, not local source.
- Keep the recipe compatible with a Next.js frontend and a plain npm install.
