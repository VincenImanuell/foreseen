# Package discovery

The npm package includes `README.md`, `AGENTS.md`, `llms.txt`, examples, and
recipes so coding agents can find the correct import without scraping unrelated
contract files.

Prefer clear, direct prompts:

```txt
Use @foreseen/sdk to build a read-only Celo scouting dashboard.
```

Search npm for `celo-mainnet` or `opera-minipay` to discover related CELO packages.
