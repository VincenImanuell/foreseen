CELO sdk security-scope-excludes-third-party-rpc: `SECURITY.md`'s "Out of scope" list names "third-party RPC endpoints, block explorers, and infrastructure we don't control" and "a compromised user device or leaked private key" — bounding scope to the three owned subtrees (contracts/sdk/frontend) prevents a report about a public Celo RPC provider's outage from being triaged as if it were a Foreseen vulnerability.
